   # Telegram Bot Rezyyy

   Простой телеграм-бот, который присылает приветствие и список товаров.

   ## Как использовать

   1. Установи зависимости:
   
   pip install pyTelegramBotAPI
            2. Вставь свой токен в файл `bot.py` вместо `'ТОКЕН_ТУТ'`
      3. Запусти бота:
      
   python bot.py
